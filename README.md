# ace-bikes-data
Download Ace Bikes Data from here.
Follow the instructions below:

## Clone Repository
### Prerequisites
- Git installed
- (Optional) Python / conda if you plan to run scripts

### 1) Clone the repository
```bash
git clone https://github.com/<YOUR-USERNAME-OR-ORG>/ace-bikes-data.git
cd ace-bikes-data
```

### 2) Download the repository if clone doesn't work
1. Open the GitHub page for this repo: `ace-bikes-data`
2. Click the green **Code** button
3. Click **Download ZIP**
4. Unzip the file on your computer
5. Open the unzipped folder (this is the project directory)
