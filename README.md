# ace-bikes-data
Download Ace Bikes Data from here.
Follow the instructions below:

## Clone Repository
### Prerequisites
- Git installed

### 1) Clone the repository
```bash
git clone https://github.com/diwanashita/ace-bikes-data.git
cd ace-bikes-data
```

### 2) Download the repository if clone doesn't work
1. Click the green **Code** button
2. Click **Download ZIP** 
3. Unzip the file on your computer
4. Open the unzipped folder (this is the project directory)
